## Product Stance

- Start with a readable homestead survival loop.
- Add economics, politics, skills, and technology only when they create concrete choices.
- Keep historical context plausible and scenario-specific.
- Treat independent agents as the long-term design center: player and AI should share the same
  basic action model.
