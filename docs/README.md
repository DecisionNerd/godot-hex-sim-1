# Documentation

**v1:** farm management sim on a hex map (~10 m tiles). **Later:** county scale (spatial buckets).

## Playable now

- Days, seasons, weather
- Food + seeds; wheat & barley crops
- Select plots (free walk) · plant / tend / harvest (labor) · end day
- Family eats 2 food per 7 days
