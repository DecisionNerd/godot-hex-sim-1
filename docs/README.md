# Documentation

**Valley Claim** is a plausible American frontier survival and settlement sim. The first scenario
is Homestead Act settlement; the broader design supports independent agents, resources, economics,
politics, skills, and technology diffusion across the 1540-1890 American West frame.

## Playable Now

- Daily turns, seasons, weather, and household survival pressure
- Homestead-era resources, chores, fields, shelter, and work zones
- Hex, patch, block, zone, and 3D terrain views from one map
- Box selection, rotate/zoom/pan controls, save/load, and automated tests
